🖥️ Components de l'ordinador

Aprèn les parts fonamentals d'un ordinador:

  • Processador (CPU): El "cervell" que executa instruccions
  • Memòria RAM: Emmagatzemament temporal ràpid
  • Disc dur (HDD/SSD): Emmagatzemament permanent
  • Placa base: Connecta tots els components
  • Fonts d'alimentació: Distribueix energia
  • Disipador tèrmic: Refredada del processador
💡 Consell: Comença per desmuntar un ordinador vell per veure els components de prop.

🔧 Muntatge bàsic i resolució de problemes

Passos per muntar un ordinador de forma correcta:

  • Descàrrega estàtica: usa polsera anti-estàtica
  • Instal·la la placa base a la caixa
  • Connecta la font d'alimentació
  • Instal·la el processador i la RAM
  • Afegeix el disipador tèrmic i los ventiladors
  • Prova la seqüència d'encesa (beep único = èxit)
⚠️ Important: Los components són molt fràgils. Tracta'ls amb cura i respecta les connexions.

⚙️ Instal·lació de SO i gestió d'usuaris

Configura un sistema operatiu nou:

  • Windows: descàrrega del .iso, crea USB arrancable, instal·la
  • Linux: distribucions gratuïtes (Ubuntu, Debian)
  • macOS: específic per a Apple
  • Gestió d'usuaris: crea comptes amb permisos adequats
  • Actualitzacions de sistema: mantén el SO segur
✅ Bona pràctica: Usa particions separades per a dades i sistema operatiu.

📚 Recursos útils

  • Speccy: Programa per veure les especificacions del teu ordinador
  • CPU-Z i GPU-Z: Monitorització detallada de components
  • Ventoy: Crea USB arrancables multi-SO fàcilment
  • YouTube: Canals com Linus Tech Tips per tutorials de muntatge
