📡 Internet de les Coses i dispositius wearables

L'IoT connecta dispositius físics a Internet:

  • SmartWatch: Rellotges intel·ligents amb sensors
  • SmartHome: Bombetes, termòstats i portes intel·ligents
  • Wearables: Dispositius portables (pulseres, anells, casques)
  • Sensors: Temperatura, humitat, moviment, CO2
  • Aplicacions: Salut, seguretat, confort, productivitat
💡 Consell: Molts dispositius IoT al mercat són assequibles (Xiaomi, TP-Link, etc.).

⚙️ Configuració de dispositius IoT

Passos típics per configurar un dispositiu smart:

  • Descarrega l'app del fabricant (xiaomi, Google Home, etc.)
  • Crea un compte i inicia sessió
  • Posa el dispositiu en mode aparellament
  • Connecta'l a la teva xarxa WiFi (2.4 GHz generalment)
  • Nomena'l dins l'app per organització
  • Configura automatitzacions o rutines
⚠️ Important: Revisa els permisos que sol·licita l'app. No tots els dades necessiten accés a micròfon o localització.

🤖 Arduino: Connexió de sensors

Arduino permet crear projectes IoT personalitzats:

  • Placa Arduino: Microcontrolador programable
  • Sensors: DHT22 (temperatura/humitat), HC-SR04 (distància), etc.
  • Actuadors: LED, motors, altaveu, pantalla LCD
  • Programació: C++ simplificat en Arduino IDE
  • Projectes: Estació meteorològica, semàfor smart, planta intel·ligent
✅ Bona pràctica: Comença amb projectes simples (llegir sensor, encendre LED) abans de fusionar-los.

📚 Recursos útils

  • Arduino Official: arduino.cc amb tutorials i documentació
  • Tinkercad Circuits: Simula Arduino online sense hardware
  • Home Assistant: Plataforma per centralitzar tots els smartdevices
  • Node-RED: Automatitzacions visuals sense codi
