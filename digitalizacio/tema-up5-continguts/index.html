📄 Documents i presentacions professionals

Crea documents visuals i ben estructurats:

  • Processadors: Google Docs, Microsoft Word, LibreOffice Writer
  • Presentacions: Google Slides, PowerPoint, Impress
  • Disseny: Canva per documents atractius sense experiència
  • PDF: Format universal per preservar formatge
  • Infografies: Piktochart, Infogram para visualitzacions
💡 Consell: Usa paletes de colors coher ents i fonts llegibles (sans-serif per a web).

🌐 Creació d'aplicacions web senzilles

Comença a crear pàgines web interactives:

  • HTML: Estructura del contingut (títols, paràgrafs, botons)
  • CSS: Estil visual (colors, layouts, animacions)
  • JavaScript: Interactivitat (clicks, validades, efectes)
  • No-code: Wix, Webflow per a no programadors
  • Frameworks: React, Vue, Angular (avançat)
✅ Bona pràctica: Comença amb templates, després personalitza amb CSS.

🥽 Introducció a Realitat Virtual i Augmentada

Explora les tecnologies immersives:

  • Realitat Virtual (VR): Entorn 360° completament digital (Meta Quest, HTC Vive)
  • Realitat Augmentada (AR): Digital superposat al món real (Instagram filters, Pokémon GO)
  • Apps: Unity, Unreal Engine per desenvolupament VR/AR
  • WebXR: VR i ar directament al navegador
  • Casos d'ús: Marc mèdica, educació, entreteniment, arquitectura
🔮 Opcional: Explora Spatial.io, Mozilla Hubs per col·laboració virtual.

📚 Recursos útils

  • Codecademy: Cursos interactius de HTML, CSS, JavaScript
  • Figma: Diseño de UI/UX col·laborativa
  • Glitch o Replit: Editors online per programar web
  • 8th Wall: Crea experiències AR per a navegador
